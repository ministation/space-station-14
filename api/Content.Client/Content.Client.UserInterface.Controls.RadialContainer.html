AngularRange

Specifies the anglular range, in radians, in which child elements will be placed. The first value denotes the angle at which the first element is to be placed, and the second value denotes the angle at which the last element is to be placed. Both values must be between 0 and 2 PI radians

Remarks

The top of the screen is at 0 radians, and the bottom of the screen is at PI radians

CalculatedRadius

Radial menu radius determines how far from the radial container's center its child elements will be placed. This is dynamically calculated (based on child button count) radius, result of InitialRadius and Content.Client.UserInterface.Controls.RadialContainer.RadiusIncrement multiplied by currently visible child button count.
