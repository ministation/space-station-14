Class InternalTemperatureComponent

Entity has an internal temperature which conducts heat from its surface. Requires TemperatureComponent to function.

Inheritance
System.Object
InternalTemperatureComponent
Namespace: Content.Server.Temperature.Components
Assembly: Content.Server.dll

public sealed class InternalTemperatureComponent : Component
Remarks

Currently this is only used for cooking but animal metabolism could use it too. Too hot? Suffering heatstroke, start sweating to cool off and increase thirst. Too cold? Suffering hypothermia, start shivering to warm up and increase hunger.
