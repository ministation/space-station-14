Namespace Content.Server.Objectives.Components

Classes

CarpRiftsConditionComponent

Requires that the dragon open and fully charge a certain number of rifts. Depends on to function.

CodeConditionComponent

An objective that is set to complete by code in another system. Use CodeConditionSystem to check and set this.

DieConditionComponent

Requires that the player dies to be complete.

DoorjackConditionComponent

Objective condition that requires the player to be a ninja and have doorjacked at least a random number of airlocks. Requires NumberObjectiveComponent to function.

EscapeShuttleConditionComponent

Requires that the player is on the emergency shuttle's grid when docking to CentCom.

HelpProgressConditionComponent

Requires that a target completes half of their objectives. Depends on TargetObjectiveComponent to function.

HijackShuttleConditionComponent

Objective condition that requires the player to leave station of escape shuttle with only antags on board or handcuffed humanoids

KeepAliveConditionComponent

Requires that a target stays alive. Depends on TargetObjectiveComponent to function.

KillPersonConditionComponent

Requires that a target dies or, if RequireDead is false, is not on the emergency shuttle. Depends on TargetObjectiveComponent to function.

MultipleTraitorsRequirementComponent

Requires that there are a certain number of other traitors alive for this objective to be given.

NotCommandRequirementComponent

Requires that the player is not a member of command.

NumberObjectiveComponent

Objective has a target number of something. When the objective is assigned it randomly picks this target from a minimum to a maximum.

ObjectiveBlacklistRequirementComponent

Requires that the objective entity has no blacklisted components. Lets you check for incompatible objectives.

ObjectiveLimitComponent

Limits the number of traitors that can have the same objective. Checked by the prototype id, so only considers the exact same objectives.

PickRandomHeadComponent

Sets the target for TargetObjectiveComponent to a random head. If there are no heads it will fallback to any person.

PickRandomPersonComponent

Sets the target for TargetObjectiveComponent to a random person.

PickRandomTraitorComponent

Sets the target for TargetObjectiveComponent to a syndicate agent. If there are no heads it will fallback to any person.

RandomTraitorAliveComponent

Sets the target for KeepAliveConditionComponent to a random traitor.

RandomTraitorProgressComponent

Sets the target for HelpProgressConditionComponent to a random traitor.

RoleRequirementComponent

Requires that the player's mind matches a whitelist. Typical use is checking for (antagonist) roles.

SocialObjectiveComponent

Marker component for social objectives and kill objectives to be mutually exclusive.

SpeciesRequirementComponent

Requires that the player's species matches a whitelist.

SpiderChargeConditionComponent

Requires that the player is a ninja and blew up their spider charge at its target location.

StealAreaComponent

An abstract component that allows other systems to count adjacent objects as "stolen" when controlling other systems

StealConditionComponent

Requires that you steal a certain item (or several)

StealResearchConditionComponent

Objective condition that requires the player to be a ninja and have stolen at least a random number of technologies. Requires NumberObjectiveComponent to function.

SurviveConditionComponent

Just requires that the player is not dead, ignores evac and what not.
