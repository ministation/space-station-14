Class ListingDataWithCostModifiers

Wrapper around ListingData that enables controller and centralized cost modification.

Inheritance
System.Object

Namespace: Content.Shared.Store
Assembly: Content.Shared.dll

public sealed class ListingDataWithCostModifiers : ListingData
Remarks

Server lifecycle of those objects is bound to FullListingsCatalog, which is their local cache. To fix cost changes after server side change (for example, when all items with set discount are bought up) Content.Shared.Store.ListingDataWithCostModifiers.ApplyAllModifiers is called on changes. Client side lifecycle is possible due to modifiers and original cost being transferred fields and cost being calculated when needed. Modifiers changes should not (are not expected) be happening on client.
