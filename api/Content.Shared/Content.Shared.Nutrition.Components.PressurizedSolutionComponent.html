Class PressurizedSolutionComponent

Represents a solution container that can hold the pressure from a solution that gets fizzy when aggitated, and can spray the solution when opened or thrown. Handles simulating the fizziness of the solution, responding to aggitating events, and spraying the solution out when opening or throwing the entity.
