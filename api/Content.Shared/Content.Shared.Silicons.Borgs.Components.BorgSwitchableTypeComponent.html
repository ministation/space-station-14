Class BorgSwitchableTypeComponent

Component for borgs that can switch their "type" after being created.

Inheritance
System.Object
BorgSwitchableTypeComponent

Assembly: Content.Shared.dll
Syntax
public sealed class BorgSwitchableTypeComponent : Component
Remarks

This is used by all NT borgs, on construction and round-start spawn. Borgs are effectively useless until they have made their choice of type. Borg type selections are currently irreversible.

Available types are specified in BorgTypePrototypes.
