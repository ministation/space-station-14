CheckBoth(Nullable<EntityUid>, EntityWhitelist, EntityWhitelist)

Checks whether a given entity is allowed by a whitelist and not blocked by a blacklist. If a blacklist is provided and it matches then this returns false. If a whitelist is provided and it does not match then this returns false. If either list is null it does not get checked.
